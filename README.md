# 收藏夹整理页

一个 Chrome 扩展，用独立页面读取现有收藏夹，并按文件夹分类展示成简洁的导航页。

## 当前功能

- 通过弹窗打开独立的收藏夹页面
- 读取当前 Chrome 收藏夹树并保存到本地
- 按文件夹目录分组展示收藏内容
- 以卡片形式展示网址
- 支持按标题和 URL 搜索
- 支持重新导入最新收藏夹数据

## 项目结构

- `popup.html`、`popup.js`：插件弹窗入口，用来打开主页面
- `dashboard.html`：收藏夹展示页面
- `dashboard.js`：导入、存储、搜索和渲染逻辑
- `dashboard.css`：页面样式

## 在 Chrome 中加载

1. 打开 `chrome://extensions`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择 `F:\Code\bookmark-organizer-extension`
5. 点击扩展图标，再点击“打开页面”
